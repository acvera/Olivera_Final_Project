Learnings

My takeaway from this course has been that we developed workflows with infrastructure as Code, tested on real enterprises and enhanced services, and analyzed utilities for provisioning and servicing enterprise servers. From an overview of our module 1, we establish that in prelims, our module 1 is on setting up and configuration of network via virtual machines creation. Configuring both remote and local machine to connect via SSH without using password but use public key and private key for module 2 For our module 3, i learned to install community enterprise operating system or red hat linux operating system and configured remote ssh connection from remote computer to centos/rhel-8. In the module 4 I learned how to apply ansible commands to perform changes in remote machines and also on play book in automating ansibe commands. I also learnt how to use when command playbook for various OS distribution and refactoring of code’s playbook.

In our Midterm module 6, I learned about working with a specific node. I was able to manage individual hosts, select and run specific plays, and remotely access services using playbooks. Module 7 focused on controlling files on remote servers and assigning roles in Ansible. This knowledge prepared me for module 8, 9, and 10, where I developed a workflow to install and configure Ansible as an infrastructure-as-code tool. In module 11, I gained an understanding of containerization, and I learned that Ansible can be used to create a workflow for continuous delivery. Module 12 involved building a sample web app in a Docker container. In module 13, I evaluated the benefits of cloud services, including NTP, OpenStack packages, SQL Database, Message queue, and membrane. Module 14 introduced me to OpenStack Installation Keystone, Glance, and Nova. Module 15 focused on Neutron, Horizon, and Cinder in OpenStack. Throughout the course, I encountered errors in my programs but learned to troubleshoot them through logic and trial and error. Overall, this course has been a valuable learning experience, introducing me to new concepts and techniques.
